# Farmacia Anna Domingo Delclós 🏥

Sitio web y catálogo interactivo premium para la **Farmacia Anna Domingo Delclós**, ubicada en Santa Maria de Palautordera, Barcelona, España.

Este proyecto ha sido desarrollado con un enfoque moderno, rápido y totalmente adaptable, priorizando la accesibilidad y el SEO óptimo.

---

## 🌟 Características Principales

-   **Aestética Premium y Moderna:**
    -   Modo Claro y Oscuro con transición suave de colores y persistencia en `localStorage`.
    -   Animaciones dinámicas fluidas para elementos interactivos.
    -   Glassmorphism y degradados de color inspirados en el diseño UI moderno.
-   **Catálogo de Productos Interactivo:**
    -   Filtro rápido de productos por categorías (Todos, Medicamentos Autorizados, Cuidado Personal, Bienestar).
    -   Carrito de compra virtual con actualización dinámica de cantidades y precio total.
-   **Gestión de Citas Médicas:**
    -   Formulario modal interactivo para agendar citas de salud con preselección del servicio de interés.
-   **SEO y Accesibilidad (A11y):**
    -   Estructura semántica HTML5 completa.
    -   Etiquetas meta de descripción, palabras clave y Open Graph para optimización de redes sociales.
    -   Uso de elementos con identificadores únicos, descripciones y textos alternativos aria-labels.
-   **Integración de Mapas:**
    -   Mapa interactivo de Google Maps embebido con la localización oficial.

## 🚀 Instalación y Uso Local

Al ser un sitio estático de alto rendimiento, no requiere configuración de servidores complejos. Puedes abrirlo directamente o servirlo localmente:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/Klaastyle/farmacia.git
    ```
2.  **Abrir en el navegador:**
    Haz doble clic en `index.html` o usa una extensión como Live Server en tu editor de código.
